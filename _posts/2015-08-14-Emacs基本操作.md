## 内容原数据<a id="sec-5-6" name="sec-5-6"></a>

以 #+ 开始，标记文档内容。
{% highlight lisp %}

    #切换正常分行
    #+BEGIN_VERSE

    #引用、左右均有部分缩进
    #+BEGIN_QUOTE

    #居中
    #+BEGIN_CENTER

    #代码
    #+BEGIN_SRC java
{% endhighlight %}
